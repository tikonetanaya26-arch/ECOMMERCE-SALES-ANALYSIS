# 🛒 E-Commerce Sales Analysis using SQL

## 📖 Project Overview
This project analyzes sales data of an e-commerce company using SQL Server. It demonstrates SQL concepts from basic to advanced and provides business insights using analytical queries.

---

## 🎯 Objectives
- Analyze customer purchase behavior
- Calculate total sales revenue
- Identify top-selling products
- Generate category-wise revenue
- Rank customers by spending
- Optimize SQL queries

---

## 🗄 Database Tables
- Customers
- Products
- Orders
